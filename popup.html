How to use Readio

  • Select text on page, right click and press Speed Read
  • Speed read current page (we'll try and find the correct content)
  • Open editor and paste any text

Quick Access Buttons

You can access the editor or read the current page through the quick access buttons in the bottom right corner. The Quick Access buttons will show up automatically on any page that is detected readable.

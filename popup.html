How to use Readio

  • Select text on page, right click and press Speed Read
  • Speed read current page (we'll try and find the correct content)
  • Open editor and paste any text

Quick Access Buttons

You can access the editor and the current page via the quick access buttons in the bottom right corner. It will show up on any webpage we can detect text on.
